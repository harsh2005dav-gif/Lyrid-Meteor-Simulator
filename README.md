# Lyrid Meteor Shower Simulator (v2.0)
A real-time, physics-based simulation of the Lyrid meteor shower developed during the 2026 peak.

## 🚀 Overview
This project simulates the atmospheric entry of Lyrid meteors using Python and Pygame. It serves as a technical demonstration of event-driven programming and orbital mechanics visualization.

## 🛠️ Features
- **Physics-Based Trajectories:** Simulates high-speed entry (~48km/s) using vector mathematics.
- **Audio Sonification:** Integrated `pygame.mixer` to trigger atmospheric "whoosh" sound effects synced with meteor generation.
- **Dynamic Rendering:** Real-time brightness decay and trail length simulation.

## 💻 Tech Stack
- **Language:** Python 3.13
- **Libraries:** Pygame (Multimedia integration), Math (Trajectory calculation)

## 🌌 Background
Developed as Project #12 in my engineering portfolio to explore the intersection of IT and Aerospace Science. Inspired by the Lyrid meteor shower peak in April 2026.
